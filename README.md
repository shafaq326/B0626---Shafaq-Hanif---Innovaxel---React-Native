# personal expense tracker

a simple react native app built with expo for tracking daily expenses. you can add, edit, delete expenses and see a summary of where your money is going.

## tech used

- react native with expo
- expo router for navigation
- react context api for state management
- asyncstorage for saving data locally on the device

## how to run

```bash
git clone https://github.com/shafaq326/B0626---Shafaq-Hanif---Innovaxel---React-Native.git
cd B0626---Shafaq-Hanif---Innovaxel---React-Native
npm install
npx expo start
```

scan the qr code with expo go on your phone, or press w to open in browser.

## what it does

- add expenses with a title, amount, category, date and optional notes
- edit or delete any expense (with a confirmation before deleting)
- all expenses sorted by most recent date
- summary screen showing total spent and a breakdown by category
- data is saved locally so it persists when you close the app
- input validation so nothing invalid gets saved
